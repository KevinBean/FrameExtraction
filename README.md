# FrameExtraction
A code that extract frames from video.

# Setup
 Install Python packages:
 `pip install -r requirements.txt` 
 
 # Usage
 Use the export.py
 Setup your video path in config.py.

 # Problems/Bugs
 - If you have importing errors, install the packages manually.
